Ask Your Database a Question

Important: Two-Step Process

This application uses a two-step process:

  1. Index Your Database Schema: Before you can ask questions, you must first index your database's schema. Run the following command in your terminal:
    python index_schema.py "your_database_uri"
  2. Ask a Question: Once your schema is indexed, you can use the form below to ask questions.

Make sure your database URI matches the one you used for indexing. See the README for more details.

Example for SQLite: sqlite:///sample.db
Example for PostgreSQL: postgresql://user:password@host:port/dbname
