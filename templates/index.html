Ask Your Database a Question

How to Use This Application

You have two options for providing the database schema:

  1. (Option A) Pre-index your database: Run this command once to teach the application your schema:
    python index_schema.py "your_database_uri"
    Then, just fill in the "Database Connection URI" and "Your Question" fields below.
  2. (Option B) Paste your schema directly: If you have a large number of tables, you can paste just the relevant schema information into the "Manual Schema Definition" box below.

See the README for more details.

Example for SQLite: sqlite:///sample.db
Example for PostgreSQL: postgresql://user:password@host:port/dbname

The schema should be in a specific format, with each table and column on a new line. For example: 

[table: users]
[table: users] [col: id] [type: INTEGER]
[table: users] [col: name] [type: VARCHAR]
[table: products]
[table: products] [col: product_id] [type: INTEGER]
