Ask Your Database a Question

How to Use This Application

You can choose between two models for generating SQL queries. For either model, you can optionally provide a manual schema to improve accuracy.

  • Local Model: Uses a model running on your machine via Ollama. This requires you to first index your database by running:
    python index_schema.py "your_database_uri"
  • External API: Uses a remote API to generate the query. Does not require pre-indexing.

See the README for more details.

Example for SQLite: sqlite:///sample.db
Example for PostgreSQL: postgresql://user:password@host:port/dbname
